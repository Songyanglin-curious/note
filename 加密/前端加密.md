# 前端加密

## 1、加密方式

常用的有三种加密方式如下：

1. base64
2. md5
3. sha1

## 2、sha1加密

1、vue中 ，通过

```
npm install js-sha1
```

安装;/

2、引用：

```
// 加密
import sha1 from "js-sha1";
```
